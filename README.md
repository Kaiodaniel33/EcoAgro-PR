# CoopAgro PR - Plataforma de Impacto Sustentável 🌾

## 🎯 Objetivo do Projeto
Projeto oficial desenvolvido para o Concurso Agrinho 2026. A aplicação tem como tema **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**. 
O objetivo foi desenvolver uma interface didática, interativa e sem bugs, para que produtores rurais compreendam tecnologias de manejo de solo e utilizem um **Simulador Matemático** para prever sua produtividade.

## 🚀 Como Funciona (Instruções de Uso)
1. **Navegação (One-Page):** Leia o conteúdo descendo a página. Se estiver no celular, utilize o menu sanduíche (`☰`) no topo.
2. **Modo Noturno (UX Avançada):** Clique no botão "Modo Escuro" (ou "Modo Claro") no cabeçalho. O JavaScript utiliza `localStorage` para salvar sua preferência.
3. **Calculadora:** Insira seu nome, a área da fazenda e selecione a tecnologia. O sistema processará os dados, exibirá um *loading* simulado e retornará uma mensagem personalizada ("Olá, [Nome]").

## 💻 Tecnologias Utilizadas (Nível 4 - Rigor Técnico)
* **HTML5:** Código estruturado de forma semântica (uso de `button`, `input`, `label`).
* **CSS3:** Estilização com seletores de Classes e IDs. Layout moderno usando `display: flex` e `display: grid`. Responsividade total via `Media Queries` (adaptando perfeitamente para celulares e tablets).
* **JavaScript (Vanilla):** Lógica funcional sem erros no console. O JS manipula o DOM (escondendo/mostrando `divs`), captura e processa variáveis, e injeta texto customizado. Código devidamente identado e com comentários explicativos.
